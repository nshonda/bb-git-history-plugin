Beaver Builder Git History plugin

== Description ==

This plugin saves a copy of the css and js written on beaver builder pages to a repository


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `bb-git-history.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
